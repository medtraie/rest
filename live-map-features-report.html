Live Map — شرح دقيق ومرتب لكل الخصائص

هذا المستند يغطي الخصائص الفعلية داخل شاشة Live Map مع شرح تشغيلي مفصل وأمثلة عملية، إضافة إلى صور توضيحية داخل كل قسم رئيسي.

1) البنية العامة للشاشة

منطقة الخريطة

  • خريطة تفاعلية + طبقات + markers + traces + alerts + heat zones.
  • دعم mini-map وتوجيه Follow/Center.

اللوحة الجانبية وDock

  • Feed/Table + Risk Radar + Timeline + Compare.
  • Dock لأزرار التحكم السريع اليومية.
مثال: المشغل يبدأ بالبحث، يحدد مركبة، ثم Follow أو Center لبدء التتبع الفوري.

2) Smart Focus 2.0

النظام يبني درجة أولوية لكل مركبة اعتمادًا على: السرعة + geofence + حالة GPS.

  • سرعة 110+ = إنذار قوي، 95+ = تنبيه متقدم.
  • خروج geofence يزيد الأولوية بشكل كبير.
  • GPS متأخر/Offline يضيف وزن خطورة.
  • انتقال ذكي بين المركبات الأعلى أولوية عند تفعيل Focus.
مثال: مركبة بسرعة 118 وخارج geofence تتقدم تلقائيًا على كل المركبات الأقل خطورة.

3) Layer Presets (Operations / Security / Replay)

الوضعالغرضما يتغير تلقائيًا
Operationsتشغيل يوميHeat ON، Alerts ON، Feed، Trace ON
Securityحساسية أمنية أعلىSmart Focus ON، Alerts ON، Heat OFF، Feed
Replayتحليل بعد الحدثReplay ON، Table، Alerts OFF، Heat OFF
مثال: في حادث مباشر يكفي الضغط على Security للتحول الفوري لوضع الاستجابة.

4) Adaptive Clustering + Viewport Rendering

  • Clustering يتغير حسب zoom والكثافة: بعيد = clusters، قريب = markers فردية.
  • Viewport Rendering: عرض العناصر داخل نطاق الخريطة الفعلي فقط.
  • تقليل الضغط البصري وتحسين الأداء مع الأساطيل الكبيرة.
مثال: عند التكبير لمدينة، تختفي الكتل تدريجيًا وتظهر مركبات منفصلة.

5) Incident Timeline + Jump

  • يسجل انتقالات الحالة: Moving/Stop/Offline/Online.
  • يسجل خروج geofence وتجاوز السرعة.
  • كل حدث قابل للنقر للانتقال مباشرة إلى موقعه على الخريطة.
مثال: النقر على حدث “Sortie geofence” يقوم بعمل Center للنقطة فورًا.

6) Multi-Select Compare (2–4 مركبات)

  • مقارنة فورية في: Speed / Stops / Risk / Compliance.
  • اختيار حتى 4 مركبات مع تحديث لحظي.
مثال: عند انخفاض Compliance لمركبة واحدة إلى 62% يتم تصنيفها كأولوية تحقق.

7) One-Click Triage داخل Alerts Feed

  • Mute 10m: كتم تنبيه متكرر.
  • Assign: إسناد التنبيه لمسؤول.
  • Escalate: تصعيد الحالات الحرجة.
مثال: تنبيه GPS متكرر يتم Mute، بينما خروج geofence يتم Escalate مباشرة.

8) Command Palette (Ctrl/Cmd+K)

  • بحث فوري عن المركبة بالاسم/IMEI/Plate.
  • تنفيذ أوامر مباشرة: Smart Focus، تبديل Feed/Table، Preset Security.
  • الوصول للأوامر دون التنقل اليدوي بين الأقسام.
مثال: Ctrl+K ثم كتابة جزء IMEI ثم Enter للانتقال الفوري للمركبة.

9) الأداء والاستقرار

  • Trace Decimation: تبسيط نقاط المسار حسب zoom.
  • Polling ذكي: تحديث أسرع للمركبات النشطة وأبطأ للثابتة.
  • Offline/Retry: شارة حالة + retry تدريجي تلقائي.
مثال: عند انقطاع API تظهر Retrying ثم تستأنف المزامنة تلقائيًا بعد نجاح الاتصال.

10) تحسينات UX والبصريات

  • Density-Aware Typography حسب كثافة المحتوى وحجم اللوحة.
  • Semantic Colors موحدة عبر الخريطة والبادجات والتنبيهات.
  • Motion Budget موجه للحالات الحرجة لتقليل التشتيت.
مثال: عند كثافة مركبات مرتفعة يتم تقليل الخط تلقائيًا للحفاظ على وضوح القراءة.

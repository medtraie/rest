1) نظرة عامة تنفيذية

Live Map منصة قيادة لحظية تجمع التتبع، التنبيه، المقارنة، والتحليل الزمني في شاشة واحدة. الهدف هو تقليل زمن الاستجابة وتحسين دقة القرار خلال التشغيل اليومي.

2) Smart Focus 2.0

  • ترتيب الأولوية يعتمد على: سرعة غير طبيعية + خروج geofence + انقطاع GPS.
  • فرز تلقائي للمركبات الأخطر.
  • انتقال ذكي بين الأهداف عالية الخطورة عندما يكون Focus مفعّلًا.
مثال تشغيلي: مركبة تجاوزت 110 km/h ثم فقدت GPS تظهر في أعلى Radar وتُتابع تلقائيًا.

3) Presets التشغيل

Presetالاستخدامالتأثير
Operationsتشغيل يوميHeat + Alerts + Feed + Trace
Securityمراقبة أمنيةSmart Focus + Alerts + Feed
Replayتحليل الحوادثReplay + Table + Trace
مثال: التحول الفوري إلى Security عند بدء Incident بدون إعادة ضبط يدوية للإعدادات.

4) Incident Timeline + Multi-Compare + Triage

Incident Timeline

  • يعرض تغير الحالة: Moving/Stop/Offline/Online + أحداث geofence/speed.
  • كل حدث يدعم Jump مباشر للموقع على الخريطة.

Multi-Select Compare

  • مقارنة 2–4 مركبات: Speed, Stops, Risk, Compliance.

One-Click Triage

  • Mute 10m / Assign / Escalate مباشرة من Feed.
مثال: تنبيه متكرر يتم Mute، تنبيه geofence يتم Escalate، ثم Assign للمسؤول المناوب.

5) الأداء والثبات

  • Adaptive Clustering + Viewport Rendering لتخفيف الازدحام البصري والحمولة.
  • Trace Decimation لتسريع الرسم في المسارات الطويلة.
  • Polling Strategy حسب النشاط ودرجة المخاطر.
  • Offline/Retry UX بشارة حالة واتصال متدرج حتى الاستعادة.

6) التوصيات التشغيلية

  • استخدم Security preset أثناء المناوبات الحرجة.
  • فعّل Compare قبل اتخاذ قرار إعادة توزيع مركبات.
  • استعمل Replay + Export/Record لتوثيق الحوادث.
  • اعتمد على PDF المرفق كمرجع تدريبي للفرق الجديدة.
